# Moving4 AI System

Цель:
Построить стабильную AI систему с памятью и контролем логики.

Стек:
- n8n (оркестрация)
- Postgres (память)
- GPT / Claude / Gemini (модели)

Структура:
- workflows — логика автоматизации
- prompts — контроль поведения AI
- memory — структура памяти
- server — запуск системы

Важно:
AI не принимает решений сам.
Вся логика контролируется системой.
